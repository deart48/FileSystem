# Задание №2 — вариант 4

##Базовый класс: 
  -элемент файловой системы (свойства название, свойство на папку, в которой он находится (если корневой, то значение null), 
  -свойство указывающее местоположение (вычисляемое), 
  -статические методы копирования и переноса объекта в другую папку, 
  -абстрактное свойство, определяющее тип элемента (сделать перечисление: папка, файл),
  -абстрактное свойство, возвращающее размер).
##Производные классы: 
  -папка, 
  -файл. 
Папка должна содержать список элементов файловой системы. В производных классах реализовать свойство, определяющее их назначение (папка или файл), и размер (для папки он вычисляется по размеру элементов, которые она содержит, для файла задаётся при создании в конструкторе).
